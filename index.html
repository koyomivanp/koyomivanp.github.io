About

作る・描く・考える

はじめまして、大島橙也です。

Unity や Java を使ったゲーム制作を中心に活動しています。プログラムだけでなく、キャラクターの見た目やUIなど「必要なビジュアル」も自分で用意できるのが強みです。

AI（Cursor 等）を開発の相棒にしながら、試作から公開まで素早く回すスタイルで制作しています。完璧な絵師ではありませんが、ゲームに必要なクオリティのビジュアルは自分で仕上げられるところまで伸ばしています。
